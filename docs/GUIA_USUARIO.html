Panel Principal

El centro de mando neurálgico del sistema. Este módulo proporciona una visión 360° de la operación diaria sin necesidad de navegar por submenús complejos.

Utiliza visualizaciones dinámicas para mostrar el flujo de caja, el rendimiento de los meseros y la popularidad de los productos. Es la herramienta ideal para gerentes que necesitan tomar decisiones basadas en datos reales en cuestión de segundos.

  • Métricas Financieras: Seguimiento en tiempo real de ventas brutas, impuestos y utilidades estimadas.
  • Estado de Ocupación: Conteo rápido de cabañas ocupadas versus disponibles para una gestión de hospitalidad eficiente.
  • Análisis de Ventas: Gráficas comparativas que muestran las horas pico de consumo durante el día.
  • Ranking de Personal: Visualización del desempeño del equipo de ventas y meseros.

Punto de Venta (POS)

Diseñado para la velocidad y la precisión. La interfaz de ventas permite procesar pedidos complejos en segundos mediante un sistema de selección táctil optimizado.

El sistema maneja automáticamente la lógica de impuestos, descuentos y, lo más importante, la integración con el sistema de hospedaje para cargos directos a la cuenta del huésped.

  • Categorías Inteligentes: Separación visual entre Cocina, Tienda y Servicios de Aventura.
  • Gestión de Cabañas: Permite asignar consumos a una habitación específica vinculando el nombre del huésped titular.
  • Tickets con QR: Generación automática de vales únicos para servicios como Tirolesa o Cabalgatas.
  • Modos de Pago: Flexibilidad total entre efectivo, tarjeta o crédito interno del hotel.

Gestión de Cocina (KDS)

Elimina el papel y los errores de comunicación. El Kitchen Display System (KDS) recibe instantáneamente los pedidos del POS y los organiza por prioridad y tiempo de llegada.

La cocina puede gestionar el flujo de trabajo de manera visual, marcando platos listos que notifican automáticamente al personal de servicio para su entrega.

  • Orden de Comandas: Sistema de tarjetas dinámicas que cambian de color según la urgencia del pedido.
  • Control de Producción: Posibilidad de ver el detalle exacto de cada plato y sus notas especiales.
  • Historial de Despacho: Registro de tiempos de preparación para auditar la eficiencia de la cocina.
  • Sincronización Total: Cada cambio en cocina se refleja en el estado global del pedido.

Validación de Tickets

Un módulo crítico para las actividades de aventura. Este sistema permite a los encargados de cada atracción (Gotcha, Caballos, Tirolesa) verificar la legitimidad de un servicio pagado.

Al escanear el QR del ticket, el sistema consulta en tiempo real si el servicio ha sido pagado y si ya fue utilizado, evitando fraudes o duplicidad de uso.

  • Escáner Inteligente: Compatible con cámaras de dispositivos móviles o escáneres láser de mano.
  • Seguridad Anti-Fraude: Marcado instantáneo de "Usado" para prevenir que un ticket sea compartido.
  • Feedback Visual: Alertas de gran tamaño (Verde/Rojo) para una operación rápida en campo.
  • Bitácora de Uso: Registro de qué empleado validó qué servicio y a qué hora exacta.

Gestión de Huéspedes

Mantén un control detallado de quién está en tus instalaciones. Este módulo centraliza la información de los visitantes, sus fechas de estancia y su historial de consumos.

Es la base del sistema de confianza del POS, permitiendo que los huéspedes disfruten de los servicios sin necesidad de cargar efectivo, pagando todo al finalizar su estancia.

  • Registro de Estancia: Captura de datos del titular y fechas de check-in/check-out.
  • Estado de Cuenta: Visualización en tiempo real de todos los cargos acumulados por la habitación.
  • Filtros Dinámicos: Búsqueda rápida por nombre, cabaña o estatus de pago.
  • Corte de Cuenta: Facilita el proceso de checkout al consolidar todas las ventas del POS en un solo saldo.

Control de Cabañas

La vista operativa de tu inventario de hospedaje. Aquí puedes gestionar la disponibilidad de cada unidad, asignar huéspedes y monitorear el mantenimiento.

El sistema permite una gestión visual intuitiva del estado de cada cabaña, facilitando la coordinación entre recepción y el equipo de limpieza.

  • Mapa de Disponibilidad: Indicadores visuales de cabañas Libres, Ocupadas o en Mantenimiento.
  • Asignación de Cargos: Acceso rápido para agregar consumos especiales directamente a la unidad.
  • Detalle de Ocupante: Información inmediata sobre quién ocupa la cabaña y sus preferencias.
  • Historial de Uso: Registro de rotación y frecuencia de uso por unidad.

Inventario y Catálogo

El motor que alimenta al POS. En este módulo se definen todos los productos, platillos y servicios que el negocio ofrece al público.

Permite un control granular sobre las existencias de productos físicos (tienda) y la configuración de servicios ilimitados (actividades), con alertas automáticas de reabastecimiento.

  • Gestión de Stock: Control de unidades disponibles con alertas de stock bajo personalizables.
  • Precios y Categorías: Organización lógica para facilitar la búsqueda en el punto de venta.
  • Servicios de Aventura: Configuración especial para ítems que generan tickets de validación QR.
  • Editor Rápido: Actualización de precios y nombres en tiempo real para todo el sistema.

Historial de Ventas

La herramienta definitiva para la transparencia financiera y auditoría. Este módulo almacena cada transacción realizada desde la instalación del sistema.

Permite rastrear discrepancias, reimprimir tickets perdidos y analizar el flujo de caja histórico para proyecciones de crecimiento.

  • Búsqueda Avanzada: Filtros por fecha, método de pago, mesero o número de transacción.
  • Reimpresión de Tickets: Generación de duplicados exactos con todos los detalles originales.
  • Desglose Detallado: Vista completa de cada ítem vendido en una transacción específica.
  • Resumen de Auditoría: Control de cancelaciones y modificaciones para prevenir fugas de capital.
