El Otro Rollo POS

Sistema de Gestión Integral Local-First

1. Visión Estratégica

El Otro Rollo POS es una solución disruptiva para el sector hospitality en entornos con conectividad limitada. Utilizando una arquitectura Local-First, garantizamos que la operación nunca se detenga, eliminando la dependencia crítica de la nube mientras ofrece una UX de nivel mundial.

🚀 Innovación
Stack moderno con React 19 y Zustand, diseñado para baja latencia y alta concurrencia local.
🛡️ Resiliencia
Base de datos persistente en el navegador que sincroniza automáticamente en segundo plano.
Figura 1.1: Interfaz principal con diseño Glassmorphic Premium (Modo Claro).

2. Manual de Operación

2.1 El Punto de Venta (POS)

El núcleo del sistema permite realizar ventas en segundos. El grid de productos está categorizado dinámicamente para facilitar la búsqueda táctil.

Figura 2.1: Terminal de Venta optimizada para entornos de baja luz (Modo Oscuro).
  • Categorización Inteligente: División automática entre Cocina, Tienda y Servicios.
  • Carrito Persistente: La orden se mantiene activa incluso si se recarga la página o se cierra el navegador.
  • Pagos Flexibles: Soporte para Efectivo, Tarjeta y Carga a Cuenta de Huésped.

2.2 Kitchen Display System (KDS)

Elimina el papel. Las órdenes llegan instantáneamente a la cocina, organizadas por tiempo de espera y estado de preparación.

Figura 2.2: Monitor de Cocina mostrando órdenes entrantes y tiempos de cocción.

2.3 Gestión de Huéspedes y Pulseras

Control total de la estancia. Vincula consumos a pulseras RFID o identificadores únicos para un flujo cashless.

Figura 2.3: Panel de Huéspedes con saldos en tiempo real y estados de ocupación.

2.4 Control de Inventario Dual

Monitorea stock finito para la tienda y gestiona insumos ilimitados para la cocina en una sola vista centralizada.

Figura 2.4: Módulo de Inventario con alertas de stock bajo y gestión de categorías.

3. Inteligencia de Negocio

El sistema genera reportes detallados y permite el "Corte de Caja" instantáneo para asegurar la transparencia financiera total en cada turno.

Figura 3.1: Historial de ventas con desglose por método de pago y reporte de ingresos.

3.2 Auto-Servicio mediante QR

Genera códigos únicos por mesa permitiendo que los clientes vean el menú desde sus propios dispositivos sin descargar apps.

Figura 3.2: Generador dinámico de códigos QR para menús digitales y ordenamiento remoto.
