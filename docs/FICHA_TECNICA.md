## 🌟 Visión del Proyecto
**El Otro Rollo POS** es una solución tecnológica disruptiva diseñada específicamente para el sector de hospitalidad y entretenimiento en entornos recreativos (parques acuáticos, resorts, clubes de playa). 

El sistema resuelve el problema crítico de conectividad en áreas abiertas mediante una arquitectura **Local-First**, garantizando operatividad total sin dependencia de internet, mientras ofrece una experiencia de usuario (UX) de nivel "Fintech Modern".

---

## 🛠️ Stack Tecnológico (Versión Pro v4.0)
Hemos migrado a un stack de vanguardia para asegurar escalabilidad y rendimiento extremo:

- **Frontend**: React 19 (Última versión estable).
- **Bundler**: Vite 6.0 (Optimización de carga instantánea).
- **Estado Global**: Zustand 5.0 (Gestión de estado ligero y reactivo).
- **Animaciones**: Framer Motion 12.0 (Micro-interacciones premium).
- **Estilos**: TailwindCSS (Design System atómico y responsivo).
- **Arquitectura**: Atomic Design (Componentes reutilizables y mantenibles).

## 📦 Módulos Estratégicos

### 1. Punto de Venta (POS) Inteligente
Interfaz táctil optimizada para alta velocidad de transacciones. Incluye:
- Grid de productos con filtrado dinámico.
- Gestión de carrito persistente (resiste recargas de página).
- Cálculo automático de propinas y cambio.

### 2. Control de Inventario Dual
Sistema único que diferencia entre:
- **Área Restaurante**: Stock infinito para productos preparados.
- **Área Tienda**: Control de stock finito con alertas automáticas de escasez.

### 3. Kitchen Display System (KDS)
Digitalización completa de la cocina, eliminando tickets de papel y reduciendo tiempos de espera mediante comunicación en tiempo real entre el POS y el área de preparación.

### 4. CRM & Gestión de Huéspedes
Registro y seguimiento de clientes para fidelización y análisis de hábitos de consumo dentro del parque.

### 5. Analytics & Business Intelligence
Historial de ventas detallado con capacidad de exportación CSV (compatible con Excel) para toma de decisiones basadas en datos.
